1)kiedy powstał JavaScript (podaj datę).

JavaScript powstał 4 grudnia 1995.

2)dlaczego jest to język skryptowy

JavaScript jest językiem skryptowym, ponieważ jest uruchamiany wyłącznie w przeglądarkach.

3)gdzie jest wykonywany JS (klient lub serwer)

JavaScript jest najczęściej stosowany po stronie klienta, czyli wykonywany na komputerze użytkownika przez przeglądarkę internetową.

4)dlaczego jest JS to bezpieczne narzędzie

Np. W JS można używać funkcji do zabezpieczania zmiennych, co pozwala chronić witrynę przed atakami Cross-Site Scripting (XSS).

5)czy jest to język obiektowy , jeśli tak to dlaczego.

JS jest obiektowym językiem skryptowym, ponieważ można użyć konstruktora, lub literału obiektu, czyli notacji klamerki, aby utworzyć obiekt za pomocą jednej linii kodu.

6)wymień zalety JS:

Zaleta1 - JS jest bardzo łatwy do użytku i opanowania.

Zaleta2 - JS ma bardzo dużą społeczność więc bardzo trudno napotkać problem, którego nikt wcześniej nie rozwiązał.

Zaleta3 - JS posiada bardzo dużą i ciągle rosnącą kolekcję bibliotek.

7)Framework to:

Jest to szkielet do budowy aplikacji, który dostarcza niezbędne biblioteki i komponenty oraz definiuje strukturę i działanie danej aplikacji.

8)Do aplikacji mobilnych można użyć Framework

Framework można używać do zarówno aplikacji webowych, jak i mobilnych.

9)Aplikacje responsywne to:

Jest to strona internetowa, która nie zapisuje się na naszym urządzeniu mobilnym.

10)Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu

>input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"<

Tworzy przycisk o nazwie "przycisk", który otworzy nowe okno przeglądarki o nazwie "Nowa Strona" za pomocą funkcji "WinOpen".

11)Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu

window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");

Jest to funkcja, którą można nadać np. przyciskowi z zadania powyżej, która otworzy nowe okno z wyświetlonym plikiem "obraz.html".



toolbar=no - ukrywa standardowy pasek narzędzi

directories=no, - ukrywa przyciski katalogów

menubar=no, - ukrywa menu przeglądarki

height=280, - ustawia wysokość okna na 280 pixeli

width=160, - ustawia szerokość okna na 160 pixeli

top=200, - ustawia położenie okna względem góry ekranu na 200 pixeli

left=200 - ustawia położenie okna względem lewej strony ekranu na 200 pixeli

12)Opisz następującą instrukcję window.close()

Pozwala na zamknięcie okna przeglądarki.
